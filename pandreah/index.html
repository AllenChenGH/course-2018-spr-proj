Expanding the Hubway Network in the City of Boston

(A CS591 L1 project)

This is a visualization for the findings of a Data Mechanics Spring 2018 project. The Map displayed below was created through Folium. On the map, the blue circles represent current Hubway Stations while the red pins represent locations for new Hubway Stations that I am proposing. Clicking the red pins reveals the number of home addresses that particular station would be serving. Along with the map, the Address Check below will tell you if the address given is within walking distance of any Hubway Station and, if it is not, it will tell you if it falls in the possible Hubway Extention Area.
